# Welcome

## Introduction
Component libraries are a great way to leverage the modular and reusable
nature of UI components (React, Vue etc). It lets your team share these components across projects, save time and keep your UI consistent.

## MDX
The stories are written as .mdx-files, for a basic syntax-table, please visit [MDXJS](https://mdxjs.com/getting-started#table-of-components)
