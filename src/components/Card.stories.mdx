### Default

The default card without any props where "content goes here" is applied as a child.

<Preview>
  <Story name="Default">

### State

The states of the card. Can be used to quickly emphasize the meaning of a card.

<Preview>
  <Story name="State">
    <form>
      <Card>
        The default card which fallbacks to use the default state value
      </Card>
      <Card state="grey">The "grey"-state of the card</Card>
      <Card state="warning">The "warning"-state of the card</Card>
      <Card state="error">The "error"-state of the card</Card>
      <Card state="success">The "success" goes here</Card>
    </form>
  </Story>
</Preview>
