# Primary Button

The primary button-component is Advisas call to action button.

## Props-table

The primary button uses the props described below. It does however also take in any extra props sent in that could be used with a button tag.
For example the prop 'disabled' which is natively supported by the html button-tag.

<Props of={PrimaryButton} />

---

### Default

The default primary button.

<Preview>
  <Story name="Default">
    <PrimaryButton>{text("Label", "Default")}</PrimaryButton>
  </Story>
</Preview>

---

### Disabled

The primary button may also be conditionally disabled by passing in 'disabled' to the component.

<Preview>
  <Story name="Disabled">
    <PrimaryButton disabled>{text("Label", "Disabled")}</PrimaryButton>
  </Story>
</Preview>

---

### Compact

In cases where you might want a smaller primary button, you may pass in a className for that.

<Preview>
  <Story name="Compact">
    <PrimaryButton className="btn--compact">
      {text("Label", "Compact")}
    </PrimaryButton>
  </Story>
</Preview>

---

### Example

An example where we use the default state(enabled) and the compact size of the **_Primary Button_** to simulate a true action button.

<Preview>
  <Story name="Example">
    <PrimaryButton className="btn--compact">Accept offer</PrimaryButton>
  </Story>
</Preview>
